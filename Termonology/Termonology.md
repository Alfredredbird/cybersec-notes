# Linux
## Shell
The command prompt instance that lets the user run commands to the system OS.
## CronJobs
Commands or tasks scheduled to run on a [[Termonology#Host|Host]] machine. Can be set for any amount of time.

# Networking
## IP
Internet Protocol.
## IPv4
Internet Protocol version 4.
Example of an IPv4 address: `192.168.x.x`
## IPv6
Internet Protocol version 6.
Example of an IPv6 address: `2345:0425:2CA1:0000:0000:0567:5673:23b5`
## Port
The port number set up on your IP to listen for incoming/outgoing connections.
## OpenSSH
The most common opensource [[SSH]] client/server.
## subdomain
we can have `example.com` as our domain but a subdomain would be `admin.example.com`.

# Penetration Testing
## Penetration Testing
The study of testing or "hacking" into a system.
## Privilege Escalation
The process of enumeration and exploitation to elevate user privileges on a machine.
## Reverse Shell
A Bash or ZSH shell that gets spawned on victim machine to allow the hacker into the machine.
## SUID
(Set User ID) is a special file permission in Linux that allows a file to execute with the permissions of its owner. See [[Privilege Escalation#SUID File Search|SUID File Search]]
